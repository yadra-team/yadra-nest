# Contributing

Yadra Nest is early-stage. Contributions should start from accepted GitHub issues in the `v0.1 Platform Baseline` milestone.

## Workflow

1. Read `AGENTS.md`.
2. Create a branch from `main`.
3. Keep the change scoped to one issue.
4. Run validation before opening a pull request.
5. Include verification commands in the pull request body.

## Validation

```bash
bash scripts/verify-ai-governance.sh
```

Security vulnerabilities must not be reported through public issues.
